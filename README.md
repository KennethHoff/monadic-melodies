# Monadic Melodies

> A functional pipeline transforming the «vibe» of a slack message into music playback. Powered by Zapier triggers

This is a Umain Hexathon 2026 project

## Technologies

During this Hackathon we were organized in many pairs. Each pair had to choose a technology from a list of categories.

Afterwards, two-and-two pairs paired up to create a squad of four.

Our squad consisted of the following two technologies:
